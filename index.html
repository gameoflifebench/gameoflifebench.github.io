game-of-life bench

a static leaderboard for models trying to design long-lived starting boards in Conway's Game of Life

about the benchmark

Each model proposes an initial board on an 8x8 toroidal grid.

We run each model across benchmark seeds and keep its best submission.

The score is steps until the first repeated global state.
