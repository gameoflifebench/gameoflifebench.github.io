game-of-life bench

a leaderboard for models trying to design long-lived starting boards in Conway's Game of Life

about the benchmark

Each model is asked to propose an initial board on an 8x8 toroidal grid.

The rule system is Conway's Game of Life.

We run each model across benchmark seeds and keep its best submission.

The main score is the number of steps before the first repeated global state, capped at 1000 steps.
